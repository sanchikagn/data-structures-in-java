    Question: difference between argv[] and args

    Hi folks,

    Would appreciate anybody who can shed some light on this.

    What is the practical difference between:
    Code:
    public static void main (String[] args) {...}
    and
    Code:
    public static void main (String argv[]) {...}
    I never thought about it until now ... and now I can't stop thinking about it!

    Re: Question: difference between argv[] and args

    If you're curious about the positioning of the array brackets, Java has some unfortunate flexibility in array declaration syntax that can cause confusion, e.g.
    Code:
    String[][] 2DArray; 
String[] 2DArray[];
String 2DArray[][];
    The normal convention is to put the array brackets after the type name.
